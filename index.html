Project Overview

The Knowledge Graph based Cyber Threat Detection Framework for E-Commerce Platforms is designed to identify suspicious activities such as fraud, malicious users, and abnormal transaction behavior in online shopping systems. The system collects user activity logs and transaction data and analyzes the relationships between users, devices, and transactions using a knowledge graph model.
